> 작성일: 2026-04-10

## 1. Executive Summary

"주거 안전 AI 스코어링 + 안전 귀가 경로 내비게이션"을 하나로 통합한 서비스는 **국내외 모두 존재하지 않는다.** 해외에는 동네 안전 스코어링(SaferHood AI, NeighborhoodScout)과 안전 귀가 모니터링(WalkSafe, Citizen, Life360)이 각각 존재하지만, 두 기능을 결합한 단일 서비스는 없다. 한국에서는 공공 서비스(안심이 앱, SS존)만 존재하며, 민간 수익화 서비스는 전무하다.

글로벌 개인 안전 앱 시장은 빠르게 성장 중이며, 해외 서비스들은 **5가지 수익화 모델**(프리미엄 구독, 데이터 API 라이선싱, 리포트 건별 판매, 기업용 SaaS, 광고·파트너십)을 통해 검증된 매출을 만들어내고 있다.

## 2. 해외 경쟁 서비스 상세 분석

### 2.1 서비스 유형별 분류

해외 유사 서비스는 크게 3가지 유형으로 나뉜다:

| 유형 | 대표 서비스 | 핵심 기능 | 안전귀가Navi와의 차이 |
|---|---|---|---|
| **A. 가족/개인 위치 안전** | Life360, bSafe | 위치 공유, 긴급 알림, 귀가 모니터링 | 안전 스코어링 없음, 경로 추천 없음 |
| **B. 동네 안전 스코어링** | SaferHood AI, NeighborhoodScout, CrimeGrade | 주소 기반 안전 점수, 범죄 데이터 분석 | 귀가 경로 기능 없음 |
| **C. 실시간 범죄/사건 알림** | Citizen, WalkSafe, Safetymap | 실시간 사건 지도, 위험지역 표시 | AI 스코어링 없음, 종합 리포트 없음 |

> **안전귀가Navi는 A+B+C를 하나로 통합하는 포지션**이며, 이 조합은 현재 시장에 존재하지 않는다.

## 3. 해외 vs 한국 비교: 왜 한국에 빈 자리가 있는가

### 3.1 한국 기존 서비스 현황

| 서비스 | 운영 주체 | 핵심 기능 | 한계 |
|---|---|---|---|
| **안심이 앱** | 서울시 | CCTV 기반 귀가 모니터링, 긴급 신고, 안심귀가 스카우트 | AI 스코어링 없음, 경로 추천 없음, 데이터를 시민에게 직접 보여주지 않음 |
| **경기도 안전귀가** | 경기도 23개 시 | GPS 위치 → 보호자/CCTV 센터 전송 | 동일 한계 |
| **SS존** | 서울시 | 물리적 방범장치(도어락, 방범창 등) 설치 | 디지털 서비스 아닌 물리적 지원 사업 |

**안심이 앱 실적:**
- 누적 다운로드: 27만 건 (2024 기준) [19]
- 누적 이용: ~21.9만 건 (2023.10 기준) [19]
- 안심귀가 스카우트 누적: 36만 건+ [20]
- 2024년 전국 서비스로 확대

**SS존 사업 현황:**
- 초기 11개 자치구 → **25개 자치구 전체로 확대** [21]
- 안심홈세트 + 안심점포 **1,500곳** 지원 [21]

### 3.2 한국 부동산 앱 시장 (잠재 B2B 파트너)

|---|---|---|
| 직방 | 229만 | 1위, 호갱노노 포함 시 점유율 61.7% |
| 호갱노노 | 179만 | 직방 계열 |
| 네이버부동산 | 134만 | 네이버 플랫폼 내장 |
| 다방 | 103만 | 스테이션3 운영 |

> **이 앱들 중 어디에도 "이 집이 안전한지"를 데이터로 보여주는 기능이 없다.** 가격·면적·교통만 다룸.

### 3.3 서울시 공공데이터 API 현황

| 데이터 | API 제공 | 비고 |
|---|---|---|
| CCTV 설치 현황 | O | 안심이 CCTV 연계현황 API (위치/좌표 포함) [22] |
| 가로등 점소등 시간 | O | 전 도로 가로등 데이터 제공 [22] |
| 자치구별 범죄 현황 | O | 5대 범죄 발생 통계 Open API [22] |
| 유흥업소 현황 | O | 열린데이터광장 |
| 편의점/경찰서 위치 | O | 공공데이터포털 |

- API 1회 요청당 최대 1,000건, 인증키 발급 후 무료 이용 [22]
- **해외 서비스 대비 한국의 강점**: 공공데이터가 체계적으로 API화되어 있어 데이터 수집 비용이 매우 낮음

### 3.4 한국에서 민간 수익화가 부재한 이유

| 요인 | 설명 |
|---|---|
| **공공 서비스 존재** | "무료 서비스가 있는데 돈 내고 쓸까?"라는 인식 장벽 |
| **그러나 공공 서비스의 한계** | 데이터 기반 의사결정 도구가 아닌, 관제센터 모니터링 구조. 시민이 직접 데이터를 보고 판단할 수 없음 |
| **부동산 앱의 무관심** | 치안/안전은 부동산 앱의 핵심 기능이 아니었음 |
| **데이터 가공의 부재** | 원천 데이터는 공개되어 있지만, 이를 종합 가공해서 "안전 점수"로 만든 사업자가 없음 |
| **미국의 인종 편향 이슈 부재** | 미국은 Redfin/Trulia가 범죄 데이터 표시를 중단(인종 차별 우려) → 한국에는 이 리스크가 낮아 오히려 기회 |

**Life360이 증명한 것:**
1. **"불안 → 안심"의 감정적 가치**는 매달 돈을 내게 만든다
2. 중간 가격대(Gold $14.99/월)가 전체 유료의 81%를 차지 — 너무 싸면 가치를 느끼지 못하고, 너무 비싸면 진입 장벽
3. 무료 사용자가 많아야 유료 전환 풀이 큼 → "기본 무료"는 필수
4. 구독 Gross Margin 85%+ → 한계비용 거의 0 → 스케일할수록 수익성 폭발

**검증된 사례: Citizen Protect ($19.99/월)**

핵심 가치는 "24시간 전문 에이전트" — 전직 911 오퍼레이터, EMT 출신이 실시간 대기. 폰 2회 흔들기로 즉시 연결. 이 가격대에서 **"사람이 나를 지켜보고 있다"는 심리적 안전감**이 결제를 만듦 [9].

**안전귀가Navi 적용안:**

| Free | 무료 | 주소 기본 안전 등급(상/중/하), 경로 1회 조회 |
| Standard | 4,900원/월 | 상세 안전 리포트 무제한, 시간대별 위험도, CCTV/가로등 지도 오버레이 |
| Premium | 9,900원/월 | 실시간 귀가 내비(AI 안전경로), 개인화(출퇴근 시간 반영), 귀가 모니터링 알림 |
