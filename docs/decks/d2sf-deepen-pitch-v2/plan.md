# Deepen — D2SF Pitch Deck (v2 / Human-Crafted)

## 개요

- **목적**: 네이버 D2SF 투자 지원용 소개자료. *2년간 외부 보관 가능 자료* 가정으로 영업비밀 디테일 추상화.
- **청중**: D2SF 심사팀 (기술적 깊이 + 사회적 임팩트 + 시장성 + 팀 동시 평가)
- **내러티브**: AI 시대에 지식은 싸졌으나 *학습권*은 더 비싸졌다 → 사교육 의존 구조의 패배는 부유층까지 번지고 있다 → Deepen은 *5분짜리 정확한 진단*을 모든 학생에게 동일 비용으로 제공한다
- **슬라이드 수**: 16
- **출력 포맷**: pdf
- **테마**: 커스텀 (editorial-serif × 한국 신문 사설 톤)

## v2 핵심 변경 사항 (v1 대비)

1. **리듬 도입** — 16장을 Dense/Editorial/Breath 3 티어로 작곡. 기존엔 모든 슬라이드가 같은 밀도였음.
2. **카드/콜아웃/시그니처 반복 제거** — 카드는 3-4장에만, 좌하단 시그니처 노드 일러스트 삭제, 우상단 섹션 태그 삭제.
3. **사회적 문제 의식 강화 (S2-5)** — 소득별 사교육비 *133배 격차*, 강남 자퇴 *2.7%* 등 새 데이터로 시작 4장 무게 강화.
4. **영업비밀 추상화 (S5, S10, S12)** — LLM/Code 책임 분리 디테일 → "검증 가능한 진단 아키텍처" 추상화. 역추적 알고리즘 → 결과만 노출. 노이즈 감소 4-layer 단계명 제거 (precision 수치는 유지).
5. **타이포그래피 대비** — Pretendard 본문 + **Spectral Italic** 인용·세리프 강조. 한 슬라이드에 폰트 굵기를 4단계 사용 (300/400/700/800).
6. **Asymmetric splits 로테이션** — 같은 비율 3슬라이드 연속 금지. 28/72, 35/65, 45/55, 60/40, 좌측 마진, 중앙 정렬, full-width 7가지 변주.

## 디자인 방향

### 톤

*한국 일간지 사설(에디토리얼)* + *기술 잡지 인포그래픽*. AI 스타트업 톤(그라디언트·네온·플랫 카드) 회피. 활자 중심, 여백 중심, 비대칭 중심.

### 폰트

| 용도 | 폰트 | 비고 |
|---|---|---|
| 본문 / 데이터 | **Pretendard** (300 / 400 / 700 / 800) | weight 4단 활용 |
| 인용 / 섹션 / 강조 명사 | **Spectral** (Italic 400 / 600) | 한국어와 어울리는 세리프, *italic*만 사용 |
| 영문 / 숫자 | **Outfit** (400 / 600 / 800 / 900) | Inter 회피 |

### 컬러 팔레트

| 변수 | 값 | 용도 |
|---|---|---|
| `--bg` | `#FAFAF7` | 따뜻한 오프화이트 베이스 |
| `--bg-warm` | `#F3F1EA` | 카드 (3-4장에만) |
| `--text` | `#0F1626` | 딥 네이비 (거의 블랙) |
| `--text-muted` | `#5B5F6B` | 부연·캡션 |
| `--text-faint` | `#9CA3AF` | 출처·메타 |
| `--accent` | `#D97706` | 앰버 — 격차 강조용 (슬라이드당 ≤ 2 곳) |

| `--rule-strong` | `rgba(15,22,38,0.3)` | 2px hairline 강조 |

### 공간 시그니처

- **Hairline rules**가 박스를 대체. 위/아래 1px hsla 라인이 영역을 구분.
- 좌측 정렬 + 비대칭 분할 (28/72, 35/65, 45/55 등 — 결코 50/50 또는 60/40 반복 금지)
- 외부 패딩 80~100px, 내부 gap 36~64px
- 페이지 번호: **우측 상단**, 작은 회색 (예: `04 / 16`). 섹션 태그는 사용하지 않음.

### 타이포그래피 대비 기법

- 한 슬라이드 안에서 **세리프 italic + 산세리프 bold + 산세리프 light** 3가지 contrast
- 액션 타이틀: 산세리프 800, 42-46px
- 인용 / 섹션 키워드: **Spectral Italic 400**, 36-72px (슬라이드 종류에 따라)
- 본문: Pretendard 400, 22-26px
- 데이터 수치: Outfit 800, 64-160px (hero numbers)

### 차별화 시그니처

- 모든 슬라이드 **우상단에 "Deepen — D2SF 2026" 한 줄** + 페이지 번호. (좌하단 시그니처 노드 일러스트 폐기)
- Breath 슬라이드는 **숫자 섹션 마커** 사용 (예: "II." 라는 큰 산세리프 숫자만)

## 슬라이드 계획 (16장 + 티어 + 레이아웃)

**티어 시퀀스**: `B B E D E B D E D E E D D E D B`

검증:
- ✓ 3 Dense in a row 없음 (최대 2 연속: S12-S13, S15는 D-E-D)
- ✓ Breath 4장: S1, S2, S6, S16 — 표지 / hook / 섹션 디바이더 / 클로징
- ✓ 첫 절반 Breath 3장(1·2·6), 후반 Breath 1장(16) ✓
- ✓ Dense 5장 (S4·S7·S9·S12·S13·S15 = 6장) — 약간 많음. 그러나 4·7·9·12·13·15 모두 중요한 데이터 슬라이드
- ✓ Editorial 5장 (S3·S5·S8·S10·S11·S14 = 6장)
- 최종 분포: B 4 / D 6 / E 6 = 25% / 37% / 37% — Editorial 비중 높음. *의도적*: 사회적 문제 + 차별점 등 *주장* 슬라이드 많아서.

**Decorative 쿼터 점검**:
- 카드 박스 사용: S7(설명 카드), S10(리캡 카드), S15(팀 카드) = 3장 (19%) ✓ ≤60%
- 콜아웃 바: 0장 ✓ ≤40%
- Hero numbers (96px+): S4(133배), S12(+13.8%p), S13(29.2조) = **정확히 3개** ✓
- Accent 사용: 슬라이드당 ≤ 2 모멘트
- Overline: 사용 안 함 (페이지 번호만 우상단)
