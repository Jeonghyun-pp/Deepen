# Deepen — D2SF Pitch Deck

## 개요
- **목적**: 네이버 D2SF 투자 지원용 소개자료 (Pitch Deck)
- **청중**: D2SF 심사팀 (기술적 깊이 + 시장성 + 팀 경쟁력 동시 평가)
- **내러티브**: AI 시대에 지식 가격은 떨어졌으나 학습 격차는 오히려 벌어지고 있다 (S) → 기존 학습 서비스는 약점을 단원 단위로만 보기 때문에 학생의 진짜 결손을 짚지 못한다 (C) → Deepen은 유형 단위 추적 + 이전 학년 결손 역추적으로 *5분짜리 강사 설명을 살 수 없는 학생* 에게도 동일한 진단을 제공한다 (R)
- **슬라이드 수**: 17
- **출력 포맷**: pdf
- **테마 모드**: 라이트 (오프화이트 베이스)
- **테마 프리셋**: 커스텀 — Deepen 톤 (editorial-serif 변형)

## 디자인 방향

- **톤**: *교과서의 차분함 + 데이터 저널의 명료함*. AI 스타트업 특유의 그라디언트/네온 톤은 피하고, 신뢰감 있는 에디토리얼 톤. 한국어 가독성 우선.
- **디스플레이 폰트**: **Pretendard ExtraBold (800)** — 한글 가독성 최상, 디지털·인쇄 모두 안정. 동시대 한국 스타트업 디자인의 *de facto* 표준이지만 weight 800 + 자간 -0.02em으로 차별화.
- **본문 폰트**: **Pretendard (400/500/600)** + 영문/숫자는 **Outfit (400/600/800)** — Inter 회피, Outfit은 기하학적이지만 따뜻함이 있어 교육 톤과 어울림.
- **색상 팔레트**:
  - 배경: `#FAFAF7` (따뜻한 오프화이트, "교과서" 톤)
  - 텍스트: `#0F1626` (딥 네이비, 거의 블랙이지만 부드러움)
  - 강조 1 (핵심 메시지·격차): `#D97706` (앰버 — *"5분의 격차"* 와 약점 노드용)
  - 강조 2 (데이터·그래프): `#1E40AF` (딥 블루 — Concept 노드, 데이터 시각화)
  - 보조 (회색 노드·기존 서비스): `#9CA3AF` (중성 그레이)
  - 라이트 (배경 카드): `#F3F1EA` (오프화이트 한 단계 어두운 톤, 카드 분리용)
- **공간 시그니처**: 좌측 정렬 고정, **60/40 분할**을 기본 레이아웃으로. 큰 여백(외부 80px, 내부 60px). 액션 타이틀은 항상 슬라이드 좌상단에서 시작.
- **텍스처**: 얇은 1px 보더 라인(#E5E0D5), 노드 일러스트는 단색 채움 + 1.5px 외곽선. 그림자 사용 금지(export 아티팩트 방지).
- **차별화 요소**:
  1. 모든 슬라이드 좌하단에 작은 *Concept(파랑) – Pattern(앰버) – Item(회색) 노드 3개 + 연결선* 일러스트가 일관 배치되어 브랜드 시그니처 역할
  2. 슬라이드 번호 대신 우상단에 *섹션 명 (Situation / Complication / Resolution)* 표기로 내러티브 구조를 시각화

- **본문**: "같은 도구를 쓰지만, 시간이 지날수록 두 학생의 학습 격차는 크게 벌어집니다."
- **주요 콘텐츠**:

- **서포팅 섹션**: 하단 임팩트 로우 — *"사교육 참여율 80%, 고1인당 월평균 77.2만원"* (출처 통계청 2024)
